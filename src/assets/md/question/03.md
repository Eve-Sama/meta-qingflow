## 简述
相信前面2题对你来说都是小case, 绝大部分翻译确实就是那么简单. 但是极少数时候, 确实会遇到一些比较难翻译的情况. 

## 需求
轻流的表单引擎中, 有一些基础字段, 在本需求中只举例了三个字段. 这些字段的名称是**记录在前端的**. 为了方便, 在本题代码中我给出了一个字符串数组, 你需要对其做国际化改造, 翻译如下.

| 关键词 | 中文 | 英文 |
| ---- | ---- | ---- |
| text | 单行文字 | Text |
| date | 日期 | Date |
| radio | 单项选择 | Radio |

以上翻译内容已写入配置文件, 直接使用即可.
